The following list of karbytes "micro directory" names denotes each set of sub-index URLs which karbytes saved every file URL of successfully to the WayBack Machine (using the Save Page Now web UI with the "save outlinks" option selected) during the time interval starting at 5:01PM Pacific Standard Time on 05_OCTOBER_2025 and ending at 4:59PM Pacific Standard Time on 06_OCTOBER_2025.

Note that, if a particular WayBack Machine Save Page Now save operation for a particular file URL or a particular sub-index URL failed upon the first attempt, karbytes repeatedly spammed the WayBack Machine with that same Save Page Now operation until the operation was successful.

Note that sometimes the WayBack Machine removes or fails to immediately display particular URL snapshots. Hence, karbytes' goal is to save each file URL of karbytes to WayBack as many times as possible during karbytes' lifespan (and ideally each (final draft version of each) file of karbytes will be saved to the WayBack Machine at least one hundred times).

https://karbytessupporter.github.io/karbytes_wayback_machine_saves_precision_batches/

https://web.archive.org/save
